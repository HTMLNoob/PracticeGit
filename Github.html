What is this?

9/17/2017

For awhile, I have been avoiding Git simply because I couldnt understand it. After giving it an actual try, It seems so simple to me now xD. Like an "aha!" moment just popped into my head. Anyway, I decided to make this as simple as possible with just a few styles. Then a few styles turned into a few more styles and now you have this. Now, why did I push myself to learn Git? I recently decided to sign up for the Chingu Cohorts. I honestly wasnt expecting to get in, but somehow they accepted my application. When I saw the prerequisites for the cohort was Git/Github, I knew I had to learn it.

At first, I was tempted to just "cheat" around it by using Github Desktop. Moreover, I actually ended up downloading it :/. But then I thought, Git is something I must learn and what am I learning from a user interface when I could be learning the language behind it? I decided to learn Git even if I didnt like it or enjoy it. I made a simple repository to mess around with, then the repository turned into a simple site. Now here I am writing this post that I doubt anyone will ever read. Nevertheless, I am enjoying Git, it makes so much sense. Now that I more or less understand it, I finally see the actual usefulness of it. Anyway, I hoped you enjoyed reading this. I might add something later on instead of this latin stuff.
